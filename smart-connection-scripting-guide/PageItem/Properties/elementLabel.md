```javascript
PageItem.elementLabel;
```

### Access

_read/write_

### Parameters

**Return value** _string_

The name of the Element Label.

## Description

The Element Label assigned to the page item. When assigning to
a text frame, all threaded text frames will get the same Element
Label.
