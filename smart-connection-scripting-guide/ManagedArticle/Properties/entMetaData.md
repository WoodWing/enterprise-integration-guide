## Syntax

```javascript
ManagedArticle.entMetaData;
```

### Access

*readonly*

### Parameters

**Return value** *EntMetaData*

An EntMetaData object.

## Description

The `entMetaData` property is used to get the Enterprise metadata associated with the Article.
