## Syntax

```javascript
Dossier.items;
```

### Access

*readonly*

### Parameters

**Return value** *string[]*

A string of Enterprise object IDs.

## Description

The `items` property is used to get a list of IDs of Enterprise objects which are residing in the current Dossier.
