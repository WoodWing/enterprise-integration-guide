## Syntax

```javascript
EntMetaData.set(key,value);
```

### Parameters

**key** *string* 

The property key.

**value** *string, Array of string, number, Array of number, boolean or Array of boolean*

The property value.

**Return value** *number*

## Description

The `set()` method adds or changes the passed property.
