```javascript
EntMetaData.set(key, value);
```

### Parameters

**key** _string_

The property key.

**value** _string, Array of string, number, Array of number, boolean or Array of boolean_

The property value.

**Return value** _number_

## Description

The `set()` method adds or changes the passed property.
