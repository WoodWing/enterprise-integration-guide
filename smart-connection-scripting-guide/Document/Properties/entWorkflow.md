## Syntax

```javascript
Document.entWorkflow;
```

### Access

*readonly*

### Parameters

**Return value** *EntWorkflow*

An EntWorkflow object.

## Description

The `entWorkflow` property is used to get access to the Enterprise worlkflow actions for the Document.
