## Syntax

```javascript
Document.managedLayoutModules;
```

### Access

*readonly*

### Parameters

**Return value** *ManagedLayoutModules*

A ManagedLayoutModules object.

## Description

The `managedLayoutModules` property is used to get all managed Layout Modules belonging to the Document.
