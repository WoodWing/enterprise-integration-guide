## Syntax

```javascript
Document.activeEdition;
```

### Access

*read/write*

### Parameters

**Return value** *string*

The active Edition.

## Description

The `activeEdition` property is used to get the active Edition of the current Document.
The metadata of the Document contains all possible Editions that can be set.
