## Syntax

```javascript
Document.defaultEditions;
```

### Access

*read/write*

### Parameters

**Return value** *string[]*

A list of Edition names.

## Description

The `defaultEditions` property is used to get the list of Edition names that new page items will be assigned to on creation.
