## Syntax

```javascript
Document.managedAdverts;
```

### Access

*readonly*

### Parameters

**Return value** *ManagedAdverts*

A ManagedAdverts object.

## Description

The `managedAdverts` property is used to get all managed adverts belonging to the Document.
