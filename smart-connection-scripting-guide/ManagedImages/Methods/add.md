## Syntax

```javascript
ManagedImages.add();
```

### Parameters

**Return value** *ManagedImage*

The created, or newly added, MangedImage object.

## Description

The `add()` method creates a new ManagedImage scripting object that can serve as the basis for creating a new image in Enterprise.
